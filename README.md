# *Claw-launcher*

script to launch claw research
# inatallation
--------------
go to the terminal and input: 


    cd
    mkdir clawscripts
    mv claw.sh /Users/nicmoore/clawscripts
    nano ~/.zshrc
then press enter
in the file that opens, write:


    export PATH="/Users/nicmoore/clawscripts:$PATH"

  
then hit ctrl + o
and then hit enter
and then hit ctrl +x
then enter the following:

    source ~/.zshrc

and press enter
